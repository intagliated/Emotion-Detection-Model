# Emotion-Detection-Model

In this project, dataset provided by Kaggle website is used (https://www.kaggle.com/astraszab/facial-expression-datasetimage-folders-fer2013), which consists of about 35,887 well structured 48 Ã— 48 pixel gray-scale images of faces. Each image has to be categorized into one of the seven classes that express different facial emotions. These facial emotions have been categorized as: 0=Angry, 1=Disgust, 2=Fear, 3=Happy, 4=Sad, 5=Surprise, and 6=Neutral. The given images are divided into three different sets which are training, validation, and test sets. There are about 28,709 training images, 3589 validation images, and 3589 images for testing. In order to classify the expressions, the features generated by convolution layers using the raw pixel data are used.
